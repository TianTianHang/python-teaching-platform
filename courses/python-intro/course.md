# Python 入门

欢迎来到 Python 入门课程！本专为零编程基础学员设计，将带你从零开始掌握 Python 编程的核心概念。

## 学习目标

完成本课程后，你将能够：
- 掌握 Python 基本语法和编程规范
- 理解变量、数据类型和运算符的使用
- 运用条件语句和循环控制程序流程
- 使用列表和字典组织和管理数据
- 编写函数实现代码复用和模块化

## 适合人群

- 完全没有编程经验的初学者
- 希望系统学习 Python 基础知识的学生
- 想要入门编程但不知道从何开始的学习者

## 课程大纲

- **第一章**：[Python简介与环境搭建](chapters/chapter-01-python-intro.md) - 了解Python历史、特点，搭建学习环境
- **第二章**：[变量与数据类型](chapters/chapter-02-variables-types.md) - 学习变量命名规则和基本数据类型
- **第三章**：[运算符](chapters/chapter-03-operators.md) - 掌握算术、比较和逻辑运算符
- **第四章**：[控制流 - 条件语句](chapters/chapter-04-conditionals.md) - 学习 if 条件判断语句
- **第五章**：[控制流 - 循环](chapters/chapter-05-loops.md) - 掌握 while 和 for 循环
- **第六章**：[列表](chapters/chapter-06-lists.md) - 学习列表的创建、操作和遍历
- **第七章**：[字典](chapters/chapter-07-dictionaries.md) - 掌握字典的创建、访问和操作
- **第八章**：[函数基础](chapters/chapter-08-functions.md) - 学习函数定义、参数和返回值

## 学习建议

- 每章学习时间：1-2小时
- 认真阅读每个知识点，理解概念后再继续
- 动手运行每段代码示例，观察输出结果
- 完成本章练习后再进入下一章
- 遇到问题可以反复阅读相关内容
