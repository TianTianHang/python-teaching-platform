# Python 入门

欢迎来到 Python 入门课程！本专为零编程基础学员设计，将带你从零开始掌握 Python 编程的核心概念。

## 学习目标

完成本课程后，你将能够：
- 掌握 Python 基本语法和编程规范
- 理解变量、数据类型和运算符的使用
- 熟练运用字符串、列表、元组、集合、字典等数据结构
- 运用条件语句和循环控制程序流程
- 使用列表推导式和生成器编写简洁代码
- 编写函数实现代码复用和模块化
- 进行文件读写操作
- 运用异常处理编写健壮的代码
- 理解模块与包的使用方法
- 掌握面向对象编程的基础概念

## 适合人群

- 完全没有编程经验的初学者
- 希望系统学习 Python 基础知识的学生
- 想要入门编程但不知道从何开始的学习者

## 课程大纲

- **第一章**：[Python简介与环境搭建](chapters/chapter-01-python-intro.md) - 了解Python历史、特点，搭建学习环境
- **第二章**：[变量与数据类型](chapters/chapter-02-variables-types.md) - 学习变量命名规则和基本数据类型
- **第三章**：[数字(Number)](chapters/chapter-03-numbers.md) - 深入学习整数、浮点数、复数及math模块
- **第四章**：[运算符](chapters/chapter-04-operators.md) - 掌握算术、比较和逻辑运算符
- **第五章**：[字符串详解](chapters/chapter-05-strings.md) - 学习字符串方法、格式化和切片操作
- **第六章**：[列表](chapters/chapter-06-lists.md) - 学习列表的创建、操作和遍历
- **第七章**：[元组](chapters/chapter-07-tuples.md) - 掌握元组的创建、不可变性及解包操作
- **第八章**：[集合](chapters/chapter-08-sets.md) - 学习集合操作、去重和成员检测
- **第九章**：[字典](chapters/chapter-09-dictionaries.md) - 掌握字典的创建、访问和操作
- **第十章**：[控制流 - 条件语句](chapters/chapter-10-conditionals.md) - 学习 if 条件判断语句
- **第十一章**：[控制流 - 循环](chapters/chapter-11-loops.md) - 掌握 while 和 for 循环
- **第十二章**：[列表推导式与生成器](chapters/chapter-12-comprehensions.md) - 学习列表推导式和生成器的使用
- **第十三章**：[函数基础](chapters/chapter-13-functions.md) - 学习函数定义、参数和返回值
- **第十四章**：[文件操作](chapters/chapter-14-files.md) - 掌握文件的读写操作和上下文管理器
- **第十五章**：[异常处理](chapters/chapter-15-exceptions.md) - 学习try-except异常处理机制
- **第十六章**：[模块与包](chapters/chapter-16-modules.md) - 理解模块导入、标准库和自定义模块
- **第十七章**：[面向对象基础](chapters/chapter-17-oop.md) - 学习类、对象和继承的基础概念
