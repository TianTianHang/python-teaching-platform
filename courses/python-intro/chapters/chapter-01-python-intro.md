## Python简介与环境搭建

### 章节概述

欢迎来到 Python 编程的世界！本章将介绍 Python 的历史、特点以及如何搭建学习环境。通过本章学习，你将对 Python 有初步了解，并准备好开始你的编程之旅。

### 知识点 1：什么是 Python

**描述：**
Python 是一种高级编程语言，由荷兰程序员 Guido van Rossum（吉多·范罗苏姆）于 1991 年创建。Python 的设计哲学强调代码的可读性和简洁的语法，使得开发者能够用更少的代码行表达概念。

**Python 的历史：**
- 1989年：Guido van Rossum 开始开发 Python
- 1991年：Python 第一个公开发行版发布
- 2000年：Python 2.0 发布
- 2008年：Python 3.0 发布（不兼容 Python 2）
- 2020年：Python 2 正式停止维护

**Python 的应用领域：**
- Web 开发（Django、Flask）
- 数据科学与人工智能（NumPy、Pandas、TensorFlow）
- 自动化脚本与运维
- 桌面应用开发
- 网络爬虫
- 游戏开发

### 知识点 2：Python 的特点

**描述：**
Python 之所以受欢迎，是因为它具有以下显著特点：

**关键要点：**
- **简单易学**：语法简洁清晰，适合初学者入门
- **免费开源**：可以自由使用、修改和分发
- **跨平台**：可在 Windows、Mac、Linux 等多个操作系统上运行
- **丰富的库**：拥有庞大的标准库和第三方库
- **解释型语言**：代码逐行执行，便于调试
- **面向对象**：支持面向对象编程范式

### 知识点 3：Python 的学习环境

**描述：**
学习 Python 需要一个合适的编程环境。本平台提供了内置的 Jupyter 笔记本环境，你可以直接在浏览器中编写和运行 Python 代码。

**使用 Jupyter 环境：**

访问平台的 [/jupyter](/jupyter) 页面，你将看到一个功能完整的 Jupyter 笔记本界面。在这里，你可以：

1. 创建新的笔记本文件
2. 编写 Python 代码
3. 立即运行并查看结果
4. 添加文字说明和笔记

**本地环境搭建（可选）：**
如果你想在自己的电脑上安装 Python，可以按照以下步骤：

1. 访问 [Python 官网](https://www.python.org/downloads/)
2. 下载适合你操作系统的 Python 安装包
3. 运行安装程序，记得勾选 "Add Python to PATH"
4. 安装完成后，在命令行输入 `python --version` 检查是否安装成功

### 知识点 4：第一个 Python 程序

**描述：**
按照编程传统，让我们从打印 "Hello, World!" 开始，这是每个程序员的第一个程序。

**示例代码：**
```python-exec
print("Hello, World!")
```

**解释：**
- `print()` 是 Python 的内置函数，用于在屏幕上输出内容
- `"Hello, World!"` 是一个字符串，用双引号或单引号包围
- Python 代码是逐行执行的，从上到下

让我们再看几个例子：

```python-exec
# 打印不同的内容
print("欢迎来到 Python 世界！")
print("Python 是一门有趣的编程语言")
print("让我们开始学习吧！")
```

### 知识点 5：Python 代码的基本规范

**描述：**
编写 Python 代码需要遵循一些基本规范，这些规范能让代码更易读、更专业。

**关键要点：**
- **注释**：使用 `#` 符号添加注释，注释不会被程序执行
- **缩进**：Python 使用缩进来表示代码块（通常是 4 个空格）
- **大小写敏感**：Python 区分大小写，`Print` 和 `print` 是不同的
- **一行一句**：通常一行只写一条语句

**示例代码：**
```python-exec
# 这是一个注释
print("Python 很有趣！")  # 这也是注释

# 缩进示例（我们后面会详细学习）
if True:
    print("这是缩进的代码块")
    print("缩进是 Python 的重要特征")
```
