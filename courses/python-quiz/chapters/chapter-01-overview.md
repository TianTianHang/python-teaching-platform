# Python 知识测验

本课程收集了 150 道 Python 选择题，涵盖从基础到专家级别的各个知识点。

## 题目分布

- **基础级别** (难度 1): Python 基础知识、数据类型、字符串、内置函数
- **中级级别** (难度 2): 中级概念、运算符、进阶用法
- **高级/专家级别** (难度 3): 高级编程、内部机制、专家系统

每道题目包含：
