## 题目描述

编写一个函数，打印指定大小的乘法表。

### 输入格式

函数接收一个参数：
- `size`：乘法表的大小（整数，默认值为9）

### 输出格式

函数不需要返回值，直接在控制台打印乘法表。乘法表的格式如下：

```
1×1=1
1×2=2	2×2=4
1×3=3	2×3=6	3×3=9
...
```

注意：使用制表符`\t`分隔同一行的不同列。

print_multiplication_table(9)

### 提示

1. 使用嵌套循环实现：
   - 外层循环控制行（第一个乘数）
   - 内层循环控制列（第二个乘数）
2. 注意内层循环的范围：对于第i行，内层循环范围是1到i
3. 使用`end="\t"`保持同一行的内容在同一行
4. 每行结束后需要换行

**示例代码框架：**
```python
def print_multiplication_table(size=9):
    for i in range(1, size + 1):
        for j in range(1, i + 1):
            print(f"{j}×{i}={i*j}", end="\t")
        print()  # 换行
```

### 注意事项
- 乘法表的格式必须正确：`j×i=结果`
- 使用`\t`作为分隔符，确保对齐
- 每一行结束后需要换行
- 默认参数size=9，支持自定义大小
