# Python流程控制

欢迎学习Python流程控制！本课程将教你如何使用条件语句和循环结构来控制程序的执行流程。

## 学习目标

完成本课程后，你将能够：
- 熟练使用if语句进行条件判断
- 掌握for循环和while循环的使用方法
- 理解循环控制语句break和continue
- 能够编写复杂的嵌套循环结构

## 适合人群

- 已掌握Python基础语法的学习者
- 想要学习程序流程控制的初学者
- 需要巩固条件语句和循环概念的学生

## 课程大纲

- **第一章**：[条件语句](../chapters/chapter-01-conditionals.md) - if、if-else、if-elif-else结构
- **第二章**：[循环结构](../chapters/chapter-02-loops.md) - for循环与while循环
- **第三章**：[循环控制](../chapters/chapter-03-loop-control.md) - break、continue与循环嵌套

## 为什么学习流程控制？

流程控制是编程的核心概念之一。没有流程控制，程序只能按顺序执行每一条语句。有了流程控制，程序可以：
- 根据不同条件执行不同的代码
- 重复执行某段代码直到满足条件
- 处理更加复杂的逻辑和业务需求

掌握流程控制是成为合格程序员的关键一步！
