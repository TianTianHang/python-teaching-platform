# 机器学习基础

欢迎来到机器学习基础课程！本课程将带你从零开始学习机器学习的核心概念和实用技能。

## 学习目标

完成本课程后，你将能够：
- 熟练使用 NumPy 进行高效数组运算
- 掌握 Pandas 数据处理和分析技能
- 使用 Matplotlib 创建专业的数据可视化图表
- 理解并实现线性回归、逻辑回归等经典算法
- 掌握支持向量机（SVM）和 K 近邻（KNN）算法
- 使用 scikit-learn 库构建和评估机器学习模型

## 适合人群

- 已完成 Python 入门课程的学习者
- 具备基本 Python 编程基础的学生
- 对数据科学和人工智能感兴趣的学习者
- 希望掌握机器学习基础技能的从业者

## 课程大纲

- **第一章**：[NumPy 与 Pandas 基础](../chapters/chapter-01-numpy-pandas-basics.md) - 数据科学基础工具
- **第二章**：[数据可视化与 Matplotlib](../chapters/chapter-02-matplotlib-visualization.md) - 数据可视化技术
- **第三章**：[线性回归](../chapters/chapter-03-linear-regression.md) - 监督学习入门
- **第四章**：[逻辑回归](../chapters/chapter-04-logistic-regression.md) - 分类问题基础
- **第五章**：[支持向量机](../chapters/chapter-05-support-vector-machines.md) - 强大的分类器
- **第六章**：[K 近邻算法](../chapters/chapter-06-k-nearest-neighbors.md) - 简单有效的学习算法

## 学习建议

1. **循序渐进**：按顺序完成各章节学习，每个概念都建立在前一个概念之上
2. **动手实践**：每个知识点都配有示例代码，建议亲自运行并修改实验
3. **完成练习**：每章配有练习题，巩固所学知识
4. **理解原理**：不仅要会调用 API，更要理解算法背后的原理
