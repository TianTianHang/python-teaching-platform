## 题目描述

执行以下Python代码后，输出结果是什么？

```python
result = 2 + 3 * 4
print(result)
```

### 题目内容

A. 14
B. 20
C. 11
D. 26

### 解析

**正确答案：A**

**详细解析：**

这道题考查Python运算符的优先级。

**代码执行分析：**

1. 表达式：`2 + 3 * 4`
2. 根据运算符优先级，**乘法（*）的优先级高于加法（+）**
3. 先计算`3 * 4 = 12`
4. 再计算`2 + 12 = 14`
5. 最终结果为`14`

**运算符优先级（从高到低）：**

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | `()` | 括号 |
| 2 | `**` | 幂运算 |
| 3 | `*`、`/`、`//`、`%` | 乘、除、整除、取模 |
| 4 | `+`、`-` | 加、减 |
| 5 | `==`、`!=`、`>`、`<`等 | 比较运算符 |
| 6 | `not` | 逻辑非 |
| 7 | `and` | 逻辑与 |
| 8 | `or` | 逻辑或 |

**选项分析：**
- **选项 A（正确）**：`2 + 3 * 4 = 2 + 12 = 14`，先乘后加
- **选项 B**：`(2 + 3) * 4 = 5 * 4 = 20`，这是加了括号的结果
- **选项 C**：`2 + 3 + 4 + 2 = 11`，没有对应的运算
- **选项 D**：`2 * 3 * 4 + 2 = 26`，没有对应的运算

**使用括号改变优先级：**
如果想要先执行加法，可以使用括号：
```python
result = (2 + 3) * 4  # 结果是20
```

**最佳实践：**
当表达式复杂时，建议使用括号明确运算顺序，这样代码更易读，也能避免优先级错误。

**更多示例：**
```python
# 复杂表达式
result = 2 + 3 * 4 - 5 / 2
# = 2 + 12 - 2.5
# = 14 - 2.5
# = 11.5

# 使用括号使代码更清晰
result = 2 + (3 * 4) - (5 / 2)
```
