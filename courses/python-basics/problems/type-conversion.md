## 题目描述

执行以下Python代码后，输出结果是什么？

```python
a = "123"
b = int(a) + 333
print(b)
```

### 题目内容

A. 123
B. 456
C. 报错
D. 123456

### 解析

**正确答案：B**

**详细解析：**

这道题考查Python的类型转换和算术运算。

**代码执行分析：**

1. `a = "123"`：创建一个字符串变量`a`，值为`"123"`（注意是字符串，不是数字）

2. `b = int(a) + 333`：
   - `int(a)`：使用`int()`函数将字符串`"123"`转换为整数`123`
   - `int(a) + 333`：`123 + 333 = 456`
   - 将结果`456`赋值给变量`b`

3. `print(b)`：输出`b`的值，即`456`

**类型转换详解：**
- `int("123")`：将字符串`"123"`转换为整数`123`
- 字符串必须包含有效的数字表示才能转换
- 如果字符串无法转换，如`int("abc")`，会引发`ValueError`错误

**常见类型转换示例：**
```python
# 字符串转整数
int("123")      # 123
int("45.6")     # 报错（不能直接转换包含小数点的字符串）

# 字符串转浮点数
float("3.14")   # 3.14
float("123")    # 123.0

# 数字转字符串
str(123)        # "123"
str(3.14)       # "3.14"
```

**选项分析：**
- **选项 A**：`123`只是`int(a)`的值，没有加上`333`
- **选项 B（正确）**：`int("123") + 333 = 123 + 333 = 456`
- **选项 C**：代码语法正确，不会报错
- **选项 D**：这是字符串拼接的结果，如`"123" + "333" = "123333"`，但这里使用的是加法运算
