## Python起步

### 章节概述

本章将带你进入Python编程的世界，了解Python的历史和特点，学会搭建开发环境，并编写你的第一个Python程序。通过本章学习，你将对Python有一个初步的认识，为后续学习打下坚实基础。

### 知识点 1：Python简介与环境搭建

**描述：**

Python是一门由荷兰程序员Guido van Rossum于1991年创建的高级编程语言。它的设计哲学强调代码的可读性和简洁的语法，尤其是使用空格缩进划分代码块。Python支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。

Python广泛应用于：
- Web开发（Django、Flask框架）
- 数据科学与机器学习（NumPy、Pandas、TensorFlow）
- 自动化脚本
- 科学计算
- 网络爬虫

**环境搭建：**

1. **安装Python**：访问[python.org](https://www.python.org/)下载并安装最新版本的Python（推荐3.8及以上版本）
2. **验证安装**：打开终端/命令行，输入 `python --version` 或 `python3 --version`
3. **选择IDE**：推荐使用VS Code、PyCharm或Jupyter Notebook

**示例代码：**
```bash
# 在终端中检查Python版本
python --version

# 进入Python交互式环境
python
```

**解释：**

第一条命令会显示已安装的Python版本号，如`Python 3.11.5`。第二条命令会启动Python的交互式解释器（REPL），你可以在这里直接输入和执行Python代码。在交互式环境中，输入`exit()`或按`Ctrl+D`可以退出。

### 知识点 2：第一个Python程序

**描述：**

学习编程的第一步通常是编写一个"Hello, World!"程序。这个简单的程序会输出一行文本，帮助你理解程序的基本结构和运行方式。在Python中，这只需要一行代码。

**示例代码：**
```python
# 这是你的第一个Python程序
print("Hello, World!")

# 输出中文也没问题
print("你好，Python！")

# 也可以输出数字
print(2024)
```

**解释：**

`print()`是Python的内置函数，用于将内容输出到控制台。括号中的内容可以是：
- 字符串（用单引号或双引号包围的文本）
- 数字（直接写数字即可）
- 变量（后续章节会学习）

`#`符号后面的内容是**注释**，用来对代码进行说明，不会被程序执行。添加注释是良好的编程习惯，可以帮助他人（以及未来的你）理解代码。

### 知识点 3：print()函数详解

**描述：**

`print()`函数是Python中最常用的函数之一。除了基本的输出功能，它还支持多种格式化选项，让你能够更灵活地控制输出内容。

**示例代码：**
```python
# 输出多个内容，用逗号分隔
print("我的年龄是", 18, "岁")

# 使用sep参数自定义分隔符（默认是空格）
print("2024", "01", "15", sep="-")  # 输出：2024-01-15

# 使用end参数控制结尾（默认是换行符）
print("Hello", end=" ")
print("World")  # 输出：Hello World

# 输出空行
print()  # 只打印一个换行符

# 输出转义字符
print("第一行\n第二行")  # \n 表示换行
print("他说：\"你好！\"")  # \" 表示双引号字符本身
```

**解释：**

- **逗号分隔**：`print()`可以接受多个参数，它们之间默认用空格分隔
- **sep参数**：自定义多个参数之间的分隔符
- **end参数**：指定输出结束时的字符，默认是换行符`\n`
- **转义字符**：以`\`开头的特殊字符，如`\n`（换行）、`\t`（制表符）、`\"`（双引号）

这些选项让你能够精确控制输出格式，在实际编程中非常实用。
