**难度：** 1
**是否多选：** false

以下哪个是合法的 Python 变量名？

```yaml
A: "123abc"
B: "my-variable"

D: "class"
```

**正确答案：** C

**解析：**
Python 变量命名规则：
1. 必须以字母或下划线开头
2. 不能以数字开头
3. 可以包含字母、数字和下划线
4. 不能使用 Python 关键字

选项 A 以数字开头，非法；
选项 B 包含连字符（-），非法；
选项 C 以下划线开头，合法；
选项 D 是 Python 关键字，非法。

### 题目 6：多选题示例

**类型：** choice
**难度：** 2
**是否多选：** true

**题目描述：**
关于 Python 中的函数参数，以下说法正确的是（多选）：

**选项：**
```yaml
A: "Python 支持位置参数和关键字参数"
B: "默认参数必须放在位置参数之后"
C: "可变参数（*args）必须放在关键字参数之后"
D: "关键字参数（**kwargs）必须放在最后"
```

**正确答案：** ["A", "B", "D"]

**解析：**
- A 正确：Python 支持位置参数、关键字参数、可变位置参数和可变关键字参数
- B 正确：函数定义时，默认参数必须放在位置参数之后
- C 错误：可变位置参数（*args）和可变关键字参数（**kwargs）可以混用，但有顺序要求
- D 正确：在函数参数中，**kwargs 必须放在最后
