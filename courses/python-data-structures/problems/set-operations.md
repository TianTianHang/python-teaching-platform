## 题目描述

执行以下Python代码后，输出结果是什么？

```python
A = {1, 2, 3, 5, 8}
B = {2, 4, 6, 7}
result = A - B
print(result)
```

### 题目内容

A. {1, 3, 5, 6, 8}
B. {1, 2, 3, 5, 8}
C. {2, 4, 6, 7}
D. {4, 6}

### 解析

**正确答案：C**

**详细解析：**

这道题考查集合的差集运算。

**代码执行分析：**

1. `A = {1, 2, 3, 5, 8}`：创建集合A
2. `B = {2, 4, 6, 7}`：创建集合B
3. `result = A - B`：
   - 集合的减号运算符（-）表示差集
   - A - B = A中存在但B中不存在的元素
   - A中的元素：1, 2, 3, 5, 8
   - B中的元素：2, 4, 6, 7
   - 2在两个集合中都存在，所以从A中移除
   - 结果是{1, 3, 5, 8}
4. 输出：{1, 3, 5, 8}

**集合的运算：**

| 运算 | 符号 | 说明 | 示例 |
|------|------|------|------|
| 并集 | A \| B | 两个集合的所有元素 | {1, 2} \| {2, 3} = {1, 2, 3} |
| 交集 | A & B | 两个集合共有的元素 | {1, 2} & {2, 3} = {2} |
| 差集 | A - B | A中但不在B中的元素 | {1, 2} - {2, 3} = {1} |
| 对称差集 | A ^ B | 只在A或只在B中的元素 | {1, 2} ^ {2, 3} = {1, 3} |

**集合的基本特性：**
- **无序**：元素没有固定顺序
- **不重复**：自动去重
- **可变**：可以添加、删除元素

**集合的常用操作：**
```python
# 创建集合
s1 = {1, 2, 3}
s2 = set([3, 4, 5])

# 添加元素
s1.add(4)  # {1, 2, 3, 4}
s1.update([5, 6])  # {1, 2, 3, 4, 5, 6}

# 删除元素
s1.remove(3)  # 删除指定元素（不存在时报错）
s1.discard(7)  # 删除指定元素（不存在时不报错）
s1.pop()  # 随机删除并返回一个元素

# 成员检查
print(1 in s1)  # True
print(7 in s1)  # False
```

**选项分析：**
- **选项 A**：这是A ∪ B的结果（并集）
- **选项 B**：这是原集合A
- **选项 C**：这是B - A的结果（差集）
- **选项 D**：这是A ∩ B的结果（交集）

**验证集合运算：**
```python
A = {1, 2, 3, 5, 8}
B = {2, 4, 6, 7}

print(f"A - B = {A - B}")      # {1, 3, 5, 8}
print(f"B - A = {B - A}")      # {4, 6, 7}
print(f"A ∩ B = {A & B}")      # {2}
print(f"A ∪ B = {A | B}")      # {1, 2, 3, 4, 5, 6, 7, 8}
print(f"A Δ B = {A ^ B}")      # {1, 3, 4, 5, 6, 7, 8}
```

**集合的应用场景：**
1. **去重**：`set([1, 2, 2, 3])` → {1, 2, 3}
2. **查找共同元素**：交集运算
3. **查找差异**：差集运算
4. **快速成员检查**：O(1)时间复杂度
5. **数学运算**：支持集合的数学运算
