## 题目描述

执行以下Python代码后，输出结果是什么？

```python
t = (1, 2, 3)
t[2] = 4
print(t)
```

### 题目内容

A. (1, 2, 3)
B. (1, 2, 4)
C. (1, 2, 3, 4)
D. 程序报错

### 解析

**正确答案：D**

**详细解析：**

这道题考查元组（tuple）的不可变性特性。

**代码执行分析：**

1. `t = (1, 2, 3)`：创建一个元组
2. `t[2] = 4`：
   - 尝试修改元组第3个元素的值
   - 元组是不可变序列，创建后不能修改
   - 这会抛出`TypeError`异常
3. 程序会因为错误而终止，不会执行到print语句

**元组的不可变性：**
- 元组一旦创建，就不能修改、添加或删除元素
- 元组的优点：
  - 性能更好
  - 更安全（防止意外修改）
  - 可以作为字典的键（因为不可变）
  - 可以作为集合的元素

**元组 vs 列表：**

| 特性 | 列表 | 元组 |
|------|------|------|
| 可变性 | 可变 | 不可变 |
| 语法 | `[]` | `()` |
| 修改 | `list[0] = value` | 不支持 |
| 添加 | `list.append(value)` | 不支持 |
| 删除 | `list.pop()` | 不支持 |
| 作为键 | 不能 | 可以 |
| 用途 | 可变序列 | 不变数据 |

**正确的元组操作：**
```python
# 创建元组
t = (1, 2, 3)

# 获取元素
print(t[0])      # 1
print(t[-1])     # 3

# 切片
print(t[1:])    # (2, 3)

# 连接
t2 = t + (4, 5)
print(t2)       # (1, 2, 3, 4, 5)

# 注意：元组连接会创建新元组，原元组不变
```

**选项分析：**
- **选项 A**：原元组，但代码会报错不会执行到print
- **选项 B**：如果元组可变会得到的结果
- **选项 C**：如果可以添加元素会得到的结果
- **选项 D（正确）**：尝试修改元组会抛出TypeError异常

**什么时候使用元组：**
1. 当数据不应该被修改时（如常量）
2. 当数据需要作为字典的键时
3. 当函数返回多个值时
4. 当数据量小且不需要修改时（性能稍好）
