## Python 编程基础

### 章节概述

本章将介绍 Python 的基础知识，帮助你理解 Python 的核心概念，掌握基本语法。通过学习本章内容，你将能够编写简单的 Python 程序。

### 知识点 1：Python 简介

**描述：**
Python 是一种高级编程语言，以其简洁的语法和强大的功能而闻名。它支持多种编程范式，包括面向对象、函数式和过程式编程。

**示例代码：**
```python
# Python 简单示例
print("Hello, Python!")
print("欢迎来到 Python 世界！")
```

**解释：**
这行代码使用 Python 内置的 `print()` 函数输出文本到控制台。每个 `print()` 语句都会在新的一行显示内容。

### 知识点 2：变量和基本数据类型

**描述：**
Python 中的变量不需要显式声明类型，可以直接赋值。常见的数据类型包括整数、浮点数、字符串和布尔值。

**示例代码：**
```python
# 变量赋值
age = 25
name = "张三"
height = 1.75
is_student = True

# 输出变量值
print(f"姓名：{name}")
print(f"年龄：{age}")
print(f"身高：{height}米")
print(f"是学生吗？{is_student}")
```

**解释：**
Python 使用动态类型系统，变量的类型在运行时确定。f-string（格式化字符串）是一种方便的字符串格式化方法，允许在字符串中嵌入变量。
