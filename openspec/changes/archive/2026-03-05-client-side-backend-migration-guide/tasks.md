## 规范内容摘要

### 决策矩阵

| 场景 | 推荐方式 |
|------|----------|
| 登录/注册页面 | 客户端直连 |
| 个人中心/设置 | 客户端直连 |
| 需要 SEO | 服务端代理 |
| 即时交互 | 客户端直连 |

### 代码模板

提供了三种主要模板：
1. 客户端页面改造模板
2. 登录后 Session 同步模板
3. 登出模板

### HTTP 客户端 API

- `clientHttp.get/post/put/delete/patch` - 通用请求
- `clientAuth.login/logout/getToken/isAuthenticated` - 认证相关

**中优先级**：
